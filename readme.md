# My personal webpage...

Built with
+ ECMAScript v6
+ node v6.11

There are 3 alternative layouts for this website, saved as branches.
+ topHeaderFinal
+ slimWideHeader
+ centeredLayout

To play with them:
```
$ npm i
$ npm start
  and in a separate terminal:
$ git checkout branchName
```
